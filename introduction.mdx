## What is Proletario Parásito Podcast Parser?

Proletario Parásito Podcast Parser is a .NET 8 console application that fetches, parses, and summarizes episode data from the [Proletario y Parásito](https://bsky.app/profile/did:plc:libeenmaoi6p455baualxw36) podcast RSS feed.

This tool retrieves the podcast's RSS feed, extracts structured metadata from each episode, and outputs a summary to the console. It handles a variety of title formats including numbered episodes, episodes with guests, and special entries.

## What Problem Does It Solve?

Podcast RSS feeds contain rich metadata, but it's often buried in XML format with inconsistent title structures and HTML-encoded descriptions. This parser:

- **Standardizes episode data** by extracting structured information from variable title formats
- **Identifies guest appearances** automatically, even with complex name lists
- **Extracts Simpsons references** embedded in episode descriptions
- **Provides clean, readable output** with color-coded console summaries
- **Analyzes the entire feed** to provide statistics across all episodes

## Technology Stack

The parser is built with modern .NET technologies and follows clean architecture principles:

- **Language:** C# 12 with nullable reference types enabled
- **Framework:** .NET 8 (latest LTS version)
- **XML Parsing:** `System.Xml.Linq` (XDocument)
- **Pattern Matching:** `System.Text.RegularExpressions` (compiled `Regex` for performance)
- **HTTP Client:** `System.Net.Http.HttpClient`
- **Zero External Dependencies:** Uses only .NET 8 standard library

## Architecture Overview

The application follows a simple, linear pipeline with clear separation of concerns:

```
RSS Feed (HTTP) → RssFeedService → EpisodeParserService → ConsoleOutputService
```

| Component | Responsibility |
|-----------|----------------|
| `RssFeedService` | Downloads and deserializes RSS XML into `RssEpisodeItem` objects |
| `EpisodeParserService` | Applies regex-based title parsing and description extraction |
| `ConsoleOutputService` | Formats and renders the summary with console colors |
| `EpisodeParseResults` | Holds all structured episode data, including computed properties |
| `RssEpisodeItem` | Holds raw field values mapped directly from the RSS item element |
